## 快速开始

### 1. 下载代码

```shell
git clone https://www.github.com/zzshi/mall.git
```

### 2. 安装依赖
```shell
cd mall/service
pip install -r requirements.txt
```

### 3. 安装数据库
```shell
cd mall/service
docker-compose -f docker-compose.yml up -d
```
